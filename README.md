# Endpoint-Extractor
Extract APIs from JavaScripts.

# Usage
1. Visit [extract.js](https://raw.githubusercontent.com/Rezy-Dev/Endpoint-Extractor/refs/heads/main/extract.js) and copy the code.
2. Create a bookmark (any browser), then name it 'Endpoint Extractor' and in place of URL add the code from [extract.js](https://raw.githubusercontent.com/Rezy-Dev/Endpoint-Extractor/refs/heads/main/extract.js) as `javascript:..snip...`.
3. Visit your target application and click the bookmarklet that you created. It should pop a new window with extracted endpoints.
